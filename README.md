## 代码风格

* 尽量使用 `C++11`
* 尽量去除宏定义，使用 `enum/constexpr` 等代替
* 尽量跨平台（虽然是 `Visual Studio` 的解决方案，但代码尽量使用标准库，以便将来在 `Linux` 下编译）
* 尽量写注释。。。
